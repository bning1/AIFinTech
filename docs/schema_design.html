AIFinTech Trading Schema Design

High-level PostgreSQL schema to support capital markets trades, instruments, portfolios, and CCR/margining.

Scope

  • Master instrument with per-asset sub-tables
  • Trades, positions, and portfolios
  • Counterparty data and CCR/margin handling

Core Tables (high level)

  • party (counterparties, internal desks)
  • portfolio
  • instrument / instrument_bond / instrument_equity
  • trade / trade_leg_swap / trade_option_details
  • margin_agreement

Full DDL

See ddl/schema.sql for the full create-table script.
